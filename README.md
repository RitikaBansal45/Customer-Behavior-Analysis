# Customer Behavior Analysis 

Overview

This project focuses on EDA using Python and analyzing customer transaction data using SQL. The goal is to extract meaningful business insights such as revenue trends, customer behavior, product performance, and segmentation.

Dataset
File: Customers.csv
Contains customer-level transactional data including:
Customer ID
Gender
Age
Purchase Amount
Product Category & Item
Review Ratings
Discount usage
Subscription status
Shipping type
Seasonal information

SQL Analysis
The SQL queries used in this project are available in: Customer Data Analysis.sql

Key Analyses Performed
1. Revenue by Gender
2. Shipping Type Insights
3. High-Value Discount Customers
4. Top Rated Products
5. Subscription Analysis
6. Customer Segmentation
7. Seasonal Revenue Trends
8. Top Product Category
9. Clothing Category Insights
10. Age Group Analysis

Key Insights
Revenue varies significantly across customer segments and demographics.
Certain shipping types and categories contribute more to overall revenue.
Loyal and recurring customers form a strong revenue base.
Seasonal trends highlight peak sales periods.
